## Next (Post-Launch — May-June 2026)

| Feature | Priority | Description |
|---------|----------|-------------|
| Signed attestations | High | Cryptographically signed evidence packets: "All agent activity between T1 and T2 is complete and unaltered." Suitable for SOC 2 / EU AI Act evidence. |
| Pre-packaged export profiles | High | One-click export bundles: SOC 2 packet, NIST AI RMF incident report, EU AI Act log bundle. Machine-readable for SIEMs + human-readable PDFs for auditors. |
| Forensic capture modes | Medium | Deep capture for regulated flows, summarized capture for dev/test. Cost optimization at scale. |
| Forensic retention policies | Medium | Customer-defined data-retention, redaction, and sampling policies separate from runtime access policies. |
| Design partner onboarding | High | Founder Rate (50% off Pro for 6 months) for first 5-10 customers with case study commitment. |

## Later (Q3-Q4 2026)

| Feature | Priority | Description |

| SIEM integrations | Medium | Splunk, Datadog, Panther — stream forensic events to existing security tooling. Build when enterprise customer requests. |
